# Expresiones equilibradas

Crea un programa que comprueba si los paréntesis, llaves y corchetes de una expresión están equilibrados.
- Equilibrado significa que estos delimitadores se abren y cieran en orden y de forma correcta.
- Paréntesis, llaves y corchetes son igual de prioritarios. No hay uno más importante que otro.
- Expresión balanceada: { [ a * ( c + d ) ] - 5 }
- Expresión no balanceada: { a * ( c + d ) ] - 5 }

#

Create a program that checks if the parentheses, braces, and brackets in an expression are balanced.
- Balanced means these delimiters open and close in the correct order.
-Parentheses, braces, and brackets have equal priority; none is more important than the other.
- Balanced expression: { [ a * ( c + d ) ] - 5 }
- Unbalanced expression: { a * ( c + d ) ] - 5 }
