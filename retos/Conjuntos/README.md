# Conjuntos

Crea una función que reciba dos array, un booleano y retorne un array.
- Si el booleano es verdadero buscará y retornará los elementos comunes de los dos array.
- Si el booleano es falso buscará y retornará los elementos no comunes de los dos array.
- No se pueden utilizar operaciones del lenguaje que lo resuelvan directamente.

#

Create a function that takes two arrays and a boolean, and returns an array.
- If the boolean is True, it will find and return the common elements between the two arrays.
- If the boolean is False, it will find and return the non-common elements from both arrays.
- You cannot use built-in operations of the language that directly solve the problem.
